The Scenario

You just joined VidMetrics as their new Vibe Coder Product Developer. The company builds YouTube analytics tools for enterprise creators and agencies.

The founder comes to you Friday morning with a half-baked idea scribbled on a napkin. One of their biggest clients, a major media company, wants a quick way to analyze competitor video performance without leaving the platform.

Here’s the rough concept:

“Build something where they can paste a competitor’s YouTube channel URL and instantly see which videos are crushing it this month. Make it look clean. Ship it Monday.”

No wireframes. No detailed specs. No design team. Just you, your AI tools, and the mission to ship.

The Mission
Build a working MVP of the competitor analysis tool that the founder can demo to the client Monday morning.

We are not just evaluating whether you can code.
We are evaluating whether you can think, build, ship, and elevate the product without being handheld.

Deliverables

1. Live Working Product
Deploy a functional web app that takes a YouTube channel URL and displays video performance data.

Requirements:

Must be accessible via a public URL

Must be functional and demo-ready

Should feel polished enough to show a client

2. Clean UI/UX
Design and implement an intuitive interface without any design handoffs.

It should:

Feel like a real SaaS product

Be easy to navigate

Look clean, modern, and intentional

Not feel like a developer-only prototype

3. Core Feature Set
At minimum, your app should include:

Channel input

Video list with key metrics

Sorting and filtering

Responsive design

Bonus points for:

Charts

Trending indicators

Export functionality

Any thoughtful features that improve the experience

4. GitHub Repository
Submit a well-organized codebase that clearly shows your development approach and AI-assisted workflow.

Include:

Clean project structure

Clear commit history if possible

README with setup instructions

Notes on how you approached the build

5. Loom Walkthrough
Record a 5-minute Loom video (camera on) walking through:

Your process

Key decisions

Tradeoffs you made

How you used AI tools to move fast

What you prioritized and why

This should feel like you’re presenting your work to leadership.

Technical Requirements
React or Next.js frontend

Tailwind for styling

Use YouTube API or any data source that gets the job done

Deploy on Vercel, Netlify, or similar

Mobile responsive
